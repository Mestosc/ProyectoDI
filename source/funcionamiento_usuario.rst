Guía de Funcionamiento para el Usuario
======================================

Esta guía detalla el flujo de trabajo recomendado para utilizar la aplicación de gestión de libros.

Introducción
------------

La aplicación permite llevar un control de tus lecturas, permitiéndote registrar libros, autores y el progreso de páginas leídas de cada uno.

Flujo de Trabajo Principal
--------------------------

Para utilizar la aplicación correctamente, se recomienda seguir este orden:

1. Gestión de Autores
~~~~~~~~~~~~~~~~~~~~~

Antes de añadir un libro, es recomendable asegurarse de que el autor ya existe en la base de datos.

*   Haz clic en el botón **"Añadir nuevo autor"** (o usa el atajo ``Alt+A``).
*   Introduce el nombre completo del autor.
*   Presiona **"Añadir autor"**.

*Nota: La aplicación utiliza el nombre del autor para vincularlo con los libros.*

2. Registro de Libros
~~~~~~~~~~~~~~~~~~~~~

Una vez tengas al menos un autor creado, puedes proceder a añadir libros:

*   Haz clic en **"Añadir nuevo libro"** (o usa ``Alt+A`` en la ventana principal).
*   **Título:** Introduce el nombre de la obra.
*   **Páginas Leídas:** El número de páginas que llevas actualmente (por defecto 0).
*   **Páginas Totales:** El número total de páginas del libro.
*   **Autor:** Empieza a escribir el nombre del autor. La aplicación te mostrará sugerencias de autores existentes. **Es obligatorio seleccionar o escribir un autor que ya esté registrado.**

3. Seguimiento de Progreso
~~~~~~~~~~~~~~~~~~~~~~~~~~

La tabla principal muestra un resumen de todos tus libros, incluyendo un porcentaje calculado automáticamente del progreso de lectura.

*   Para actualizar tu progreso, selecciona un libro de la lista y haz clic en **"Editar seleccionado"** (o haz doble clic sobre la fila).
*   Modifica el campo **"Páginas Leídas"** con tu nuevo avance.
*   Al guardar, el porcentaje se actualizará automáticamente en la vista principal.

4. Gestión de la Biblioteca
~~~~~~~~~~~~~~~~~~~~~~~~~~~

*   **Edición:** Puedes cambiar cualquier dato de un libro (título, autor o páginas) seleccionándolo y pulsando editar.
*   **Eliminación:** Si deseas borrar un registro, selecciónalo y pulsa **"Eliminar seleccionado"** (o ``Alt+L``). Se te pedirá una confirmación antes de proceder.

Atajos de Teclado
-----------------

La aplicación soporta mnemónicos para facilitar el uso rápido:

*   ``Alt+A``: Añadir nuevo libro (desde la ventana principal).
*   ``Alt+U``: Añadir nuevo autor (desde la ventana principal).
*   ``Alt+E``: Editar el libro seleccionado.
*   ``Alt+L``: Eliminar el libro seleccionado.
